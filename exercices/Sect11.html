Directive {$N}
[haut]
Il existe 5 types réels prédéfinis : Real, Single, Double, Extended et Comp. Ecrire une fonction qui retourne le produit de deux réels de type Comp. Compiler le programme en ligne de commande et observer le résultat.

Le compilateur en ligne de commande se nomme TPC, vous devez ajouter le chemin du répertoire /BIN de TP dans le PATH.

Voir le fichier Extend.pas

Solution : compiler en utilisant la directive $N+ : 

TPC -$N+ Extend.pas
